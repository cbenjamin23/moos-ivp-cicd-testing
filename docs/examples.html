Method

One mutation at a time

  1. Run the focused test against the unchanged MOOS-IvP checkout.
  2. Apply one intentional source edit without creating a commit.
  3. Rebuild only the affected target and capture the failing result.
  4. Restore the source and rebuild the original target before continuing.
