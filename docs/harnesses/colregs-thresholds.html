COLREGS

H02 COLREGS Thresholds

Boundary correctness tests for COLREGS classification. These cases probe below, edge, above, and mirrored geometries to catch regressions near rule-transition thresholds.

Stem Mission

Baseline scenario context

The shared COLREGS stem mission uses two vessels, ben and abe; ben is the evaluated ownship. Threshold cases reuse the same stem while moving the geometry just below, at, or above classification boundaries.
